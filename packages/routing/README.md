# @kumahq/routing

Routing utilities and Vue components (e.g. `<RouteView />` and `<RouteTitle
/>`).

## Components

- **RouteTitle** - Registers a dynamic page title with the nearest parent
RouteView.
- **RouteView** - Root route component that manages route state, URL
parameters, document title, and exposes utilities (`uri`, `route`, `can`,
`env`, `t`) to its slot.
